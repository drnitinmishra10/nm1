An Intelligent Hybrid CNN–Transformer Framework for Automated Skin Disease Detection and Classification

This research paper presents an intelligent hybrid deep learning framework for automated skin disease detection and classification using dermoscopic images. The study addresses the limitations of traditional Convolutional Neural Networks (CNNs) and transformer-based architectures by integrating both models into a unified AI-driven healthcare framework capable of improving diagnostic accuracy, contextual learning, robustness, and explainability.

Research Problem

Existing skin disease detection systems face several major challenges including limited contextual learning, high computational complexity, lack of interpretability, dataset imbalance, overfitting, and reduced classification robustness. Traditional CNN models effectively capture local lesion patterns but fail to model global spatial relationships, while transformer-based models provide strong contextual understanding but require high computational resources and large annotated datasets.

Identified Core Problem:
Existing AI-based dermatological diagnosis systems fail to simultaneously achieve accurate local feature extraction, global contextual learning, computational efficiency, interpretability, and robust multiclass classification performance.

Proposed Solution

To overcome these limitations, the paper proposes a hybrid CNN–Transformer architecture that combines CNN-based local feature extraction with transformer-based global attention learning. The framework also integrates attention-guided feature fusion and Explainable Artificial Intelligence (XAI) techniques using Grad-CAM for transparent diagnostic interpretation.

  • CNN module for local feature extraction
  • Transformer encoder for global contextual learning
  • Attention-guided hybrid feature fusion
  • Softmax-based multiclass classification
  • Grad-CAM explainability integration
  • Data augmentation and regularization strategies

Methodology

The proposed methodology includes dermoscopic image acquisition, preprocessing, augmentation, CNN-based feature extraction, transformer-based attention learning, feature fusion, classification, and explainable AI visualization. Benchmark datasets such as HAM10000 and ISIC Archive were used for evaluation.

Methodology Component Purpose
Image Preprocessing Noise removal and normalization
Data Augmentation Improve generalization and reduce overfitting
CNN Backbone Extract local lesion features
Transformer Encoder Capture global contextual dependencies
Feature Fusion Combine local and global representations
Grad-CAM Provide explainable diagnostic visualization

Experimental Results

Experimental analysis demonstrated that the proposed hybrid CNN–Transformer framework significantly outperformed existing CNN and transformer-only models across multiple evaluation metrics including accuracy, precision, recall, F1-score, and ROC-AUC.

Key Contributions

  • Development of a novel hybrid CNN–Transformer architecture
  • Attention-guided contextual feature fusion
  • Explainable AI integration using Grad-CAM
  • Improved classification robustness and generalization
  • Reduced overfitting and class imbalance effects
  • Deployment-oriented intelligent healthcare framework

Conclusion

The proposed intelligent hybrid CNN–Transformer framework successfully addressed several limitations of existing AI-based skin disease diagnosis systems. The integration of local spatial learning, transformer-based contextual modeling, and explainable AI significantly improved classification accuracy, robustness, and interpretability. The framework demonstrated strong potential for real-world deployment in intelligent healthcare systems, telemedicine platforms, mobile dermatology applications, and AI-assisted clinical decision support systems.
